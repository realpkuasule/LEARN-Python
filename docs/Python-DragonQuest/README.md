## 每章结构

每一章都是独立完整的教案，包含：

1. **冒险引入** —— 用 RPG 故事场景引出本章要解决的问题
2. **学习目标** —— 学完本章能做什么
3. **核心讲解** —— 概念 + 代码示例 + 逐行解读 + 类比
4. **AI 协作指南** —— 这个知识点在跟 AI 对话时怎么用
5. **实战练习** —— 动手改代码、加功能
6. **本章回顾** —— 一句话总结 + 下章预告

## 贯穿全书的角色

- **勇者刘老三**：你的角色，从「史莱姆克星」成长为「恶龙杀手」
- **魔法书（AI）**：你的冒险伙伴，能精准执行你的每一个指令——前提是你表达得足够清晰
- **冒险者公会**：学习编程的隐喻——登记、接任务、交任务

## 如何使用

- **自学**：按章节顺序阅读，每章动手跑代码、改代码、让 AI 帮你扩展
- **教学**：每章约 45-60 分钟，冒险引入 5min + 核心讲解 25min + 动手练习 15min
- **参考**：附录速查表在每章文件末尾，快速定位概念↔RPG 映射
